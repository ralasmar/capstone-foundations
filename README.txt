PROJECT DESCRIPTION------------------------------------------------------------
This project is a social Media web application where users share music reviews. 

TECH STACK---------------------------------------------------------------------
Javascript, node.js, SQL, HTML, CSS, express, axios

FEATURES-----------------------------------------------------------------------
1. Users can search database for an album to view reviews
2. Users can post their own reviews with album name, artist name, album cover, review body, and a star rating
3. Users can delete reviews from their page
4. Users can like/dislike react to other users' reviews
5. Users can change their profile photo
